# files.gallery 功能复刻实现计划

## 执行顺序

### Wave 1 (P0 - 核心功能) — 并行实现
1. **音频播放器** — 后端ID3 API + 前端播放列表组件
2. **文本文件编辑保存** — 后端PUT API + 前端保存按钮
3. **ZIP打包下载** — 后端ZIP API + 前端批量操作
4. **文件夹预览图** — 前端FileGrid修改

### Wave 2 (P1 - 重要功能) — 并行实现
5. **多用户权限** — 数据库迁移 + 权限中间件 + UI
6. **上传限制配置** — 后端检查 + 前端提示
7. **上传冲突处理** — 后端策略 + 前端提示
8. **批量操作增强** — 批量移动/复制

### Wave 3 (P2 - 增强功能)
9. **图片格式转换** — HEIC/PSD/TIFF支持
10. **视频/PDF缩略图** — 外部服务集成
11. **设置编辑器完善** — 管理面板
12. **诊断页面** — 系统检查UI

### Wave 4 (P3 - 锦上添花)
13. **国际化完善** — 提取硬编码字符串
14. **CSS自定义** — 自定义主题
15. **批量复制链接** — 一键复制

---

## 技术决策

### 音频播放器
- 使用已安装的 jsmediatags 解析ID3标签
- 使用已安装的 plyr 作为播放器
- 后端新增 /api/id3 端点提取元数据
- 前端新增 PlaylistPlayer 组件

### 文本文件编辑
- 后端新增 PUT /api/file 端点（覆盖写入）
- CodeEditor 和 MarkdownEditor 添加保存功能
- authMiddleware 保护

### ZIP打包下载
- 使用已安装的 jszip
- 前端打包（避免服务器内存压力）
- BulkActions 添加 "ZIP下载" 按钮

### 多用户权限
- 新增 migration 添加 permissions JSON 字段
- 权限中间件读取用户配置
- admin 面板添加用户管理

### 上传限制
- settings 表存储配置
- 上传时检查
- 前端显示限制信息

### 文件夹预览图
- FileGrid 中文件夹卡片显示第一张图片缩略图
- 使用已有的 /api/thumbnail 端点
